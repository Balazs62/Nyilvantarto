# Tesztfeladat - Anyagnyilvántartó

Ez egy WPF alapú nyilvántartó program anyagok kezelésére.

## Funkciók
- Anyagok listázása és keresése
- Új anyagok felvitele (validációval)
- 
## Hogyan indítsd el? (Telepítés nélkül)
Mivel ez a forráskód, a futtatáshoz Visual Studio szükséges:

1. Töltsd le a repót (vagy a ZIP-et) és csomagold ki.
2. Keresd meg a mappában a **`Nyilvantarto.sln`** fájlt és nyisd meg.
3. Visual Studio-ban kattints felül a zöld **Start** (Play) gombra .
